## O que é o MedXF?

O MedXF é um sistema web desenvolvido para auxiliar no processo de triagem da Síndrome do X Frágil, oferecendo uma plataforma para registro e acompanhamento inicial de informações dos pacientes. O projeto busca facilitar a coleta e visualização de dados de forma intuitiva e eficiente, além de demonstrar como soluções tecnológicas podem apoiar iniciativas voltadas à área da saúde e inclusão. Desenvolvido como atividade de extensão acadêmica, o sistema aplica conhecimentos de desenvolvimento de software em um contexto social e educacional.

## Funcionalidades

### Administrador

- Autenticação e controle de acesso
- Cadastro e gerenciamento de médicos
- Gerenciamento de pacientes
- Gerenciamento de avaliações
- Visualização de relatórios

### Médico

- Autenticação de médicos
- Gerenciamento de avaliações
- Visualização de pacientes
- Realização de novas triagens
